# WannaCryNetScanner
A simple python scanner to detect vulnerable hosts in your net


### Usage
```
python netscanner.py <ip> (192.168.0.2) or <network> (192.168.0.*)
```

### Prerequisites
```
Python >= 2.7
python-nmap >= 0.6.1 (https://pypi.python.org/pypi/python-nmap)
```
